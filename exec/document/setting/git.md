# Git 설치 및 프로젝트 셋팅

## Git 설치

```bash
$ sudo apt-get install git
$ sudo apt install git
```

## Git 버전 확인

```bash
$ sudo git --version
>>
```

git version 2.25.1

````

## Git 계정 설정

```bash
$ sudo git config --global user.name ${유저 이름}
$ sudo git config --global user.email ${유저 이메일}
````

## Git을 이용하여 프로젝트 Clone

```bash
$ sudo git clone ${클론 받을 깃 레포지토리 url} ${다운받아올 폴더명}
```
